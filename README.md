# 🎂 Animated Birthday Card

A simple and beautiful animated birthday webpage created using HTML and CSS.

## 🚀 Features
- Animated floating text
- CSS candle flame animation
- Cake layers drop-in effect
- Clean and responsive design

## 🛠️ Tech Stack
- HTML
- CSS

## ▶️ How to Use
1. Open `index.html` in a browser
2. Enjoy the animated birthday card 🎉

## 💡 Purpose
This project was created to practice basic frontend skills and CSS animations.
